📋 Google Apps Script Setup

Follow these steps to connect:

  1. Your Google Sheet should have columns: Date | Event | Amount | Category | Completed | Comment
  2. Open your sheet → Extensions → Apps Script
  3. Copy the script from the setup guide
  4. Deploy as Web App (Execute as: Me, Access: Anyone)
  5. Copy the deployment URL and paste it below

URL format: https://script.google.com/macros/s/ABC.../exec

⚠️ IMPORTANT: Add a new "Category" column (column D) after Amount!
